Understanding Markup Tags in Web Development

Markup Tag Description Possible Attributes Description of Attributes
<html> Defines the root of an HTML document. lang lang: Specifies the primary language of the content within the element.
<head> Contains metadata about the HTML document. None The tag does not have any attributes. It is used to contain meta-information about the HTML document, such as the title, metadata, scripts, stylesheets, and more.
<title> Defines the title of the document. None The title tag does not have any attributes. It is used to define the title of the HTML document, which appears in the browser's title bar or tab
<body> Contains the content of the HTML document. bgcolor, text Specifies the background color of the element. It can be defined using color names, hexadecimal color codes, or RGB values.
<h1> to <h6> Defines headings of decreasing importance. None The heading elements h1 to h6 do not have any attributes. They are used to define headings of varying levels of importance, with h1 being the highest and h6 being the lowest
<p> Defines a paragraph. None The p tag does not have any attributes. It is used to define a paragraph of text within an HTML document.
<a> Defines a hyperlink. href, target href: Specifies the URL of the linked document.
target: Specifies where to open the linked document (e.g., _blank, _self).
<img> Inserts an image. src, alt src: Specifies the URL of the image file.
alt: Specifies an alternate text for the image.
<ul> Defines an unordered list. type Specifies the type of bullet point used for the list items within the ul element. It can take the values disc (default), circle, or square, determining the appearance of the bullet points.
<ol> Defines an ordered list. start, reversed start: Specifies the starting value of the first item in the ordered list. reversed: Specifies that the numbering of the list items should be displayed in reverse order.
<li> Defines a list item. Value Specifies the value of the list item within an ordered list ol. It is used to override the default numbering of the list item and can be any positive integer.
<div> Defines a division or section. None div elements are often used in conjunction with CSS for layout purposes, allowing developers to apply styling and structure to sections of content within a webpage.
<span> Defines a section in a document for styling purposes. None span elements are commonly used to target and style individual words, phrases, or inline elements within a larger block of text.
<table> Defines a table. border, width, height, align border: Specifies the width of the border around the table or individual cells. width: Specifies the width of the table. height: Specifies the height of the table. align: Specifies the horizontal alignment of the table within its container. It can take values like left,center, or right
<tr> Defines a table row. None The tr tag does not have any specific attributes. It is used to define a row within an HTML table table
<th> Defines a table header. colspan, rowspan colspan: Specifies the number of columns that a header cell (th) should span. rowspan: Specifies the number of rows that a header cell (th) should span.
<td> Defines a table cell. Valign valign: Specifies the vertical alignment of the content within the data cell td. It can take values like top, middle, or bottom.
<form> Defines an HTML form for user input. action, method action: Specifies where to send the form data.
method: Specifies the HTTP method for sending form data (e.g., GET, POST).
<input> Defines an input control. type, name, value type: Specifies the type of input control (e.g., text, checkbox, radio).
name: Specifies the name of the input control.
value: Specifies the value of the input control.
<button> Defines a clickable button. autofocus, disabled, form autofocus: Specifies that the button should automatically get focus when the page loads. disabled: Specifies that the button should be initially disabled. form: Specifies one or more forms the button belongs to.
